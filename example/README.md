# React Flow Examples

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Installation

```sh 
npm install
```



## Start 

Starts a dev server at http://localhost:3000

```sh 
npm start
```

## Build 

```sh 
npm run build
```
